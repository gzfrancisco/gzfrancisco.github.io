gzfrancisco.github.io
=====================

## Dependecies

* Ruby 2.0+
* Bundler
* Jekill
* Sass
* Bourbon
* Foreman

## Developer mode

To run the developer mode, in terminal run foreman with all task.

    $ bundle
    $ foreman start
